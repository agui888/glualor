# glualor

A Web Framework For Gopher Lua

## required

```shell
go get github.com/yuin/gopher-lua
go get github.com/zhu327/gluatemplate
go get github.com/zhu327/gluaweb
go get layeh.com/gopher-json
```

## run

```shell
go build main.go
./main
```

<http://lor.sumory.com/>
